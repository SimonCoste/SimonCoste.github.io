Flow models V: consistency, flow maps and distillation

March 2025

Let us summarize what we've seen. Given two p0,p1p_0, p_1p0​,p1​, we found a way to define an ODE X˙t=vt(Xt)\dot{X}_t = v_t(X_t)X˙t​=vt​(Xt​) such that X0∼p0X_0 \sim p_0X0​∼p0​ and X1∼p1X_1 \sim p_1X1​∼p1​. Score-matching techniques allow us to learn vtv_tvt​ from samples. Once this is done, we can sample from p1p_1p1​ by solving the ODE. Alternatively, there is an SDE whose marginals are also ptp_tpt​ and whose drift is related to vtv_tvt​, so we could also sample by sampling from this SDE.

In practice, we can't solve the ODE or SDE exactly, so we discretize it using, say, NNN time steps t1<⋯<tNt_1 < \dotsb < t_Nt1​<⋯<tN​, using schemes like Runge-Kutta for the ODE, or Euler-Maruyama for the SDE. But then, we need NNN evaluations of the neural network that learnt vtv_tvt​, which hinders a lot the speed of the sampling processes. As a result, big models like FLUX.1 can take a few tens of seconds (on a 40GB GPU) to sample a single image.

The ODE X˙t=vt(Xt)\dot{X}_t = v_t(X_t)X˙t​=vt​(Xt​) started at x0x_0x0​ produces a flow Ψt(x0)\Psi_t(x_0)Ψt​(x0​), which is the solution of the ODE at time ttt starting at x0x_0x0​. Is it possible to directly learn this flow map Ψt\Psi_tΨt​ with a single neural network?

It turns out that the flow map can be learnt directly from the knowledge of the velocity field vtv_tvt​.

The flow map Ψs,t\Psi_{s,t}Ψs,t​ is the minimizer of the square loss

Few-steps sampling

The learnt flow map SSS can be used in various ways to sample from p1p_1p1​.

  • for super fast sampling, we can directly sample x0∼p0x_0 \sim p_0x0​∼p0​ and apply S0,1(x0)S_{0,1}(x_0)S0,1​(x0​) to get a sample from p1p_1p1​. Indeed, the first « flow map learning » method did directly learn S0,tS_{0,t}S0,t​ for all ttt and was called consistency distillation. However, it can happen that this map is not accurate: it is quite intuitive that learning how to go from sss to s+Δss + \Delta ss+Δs is easier than going directly from 000 to 111, so it can happen that S0,1S_{0,1}S0,1​ is not very accurate.

  • One can trade speed for accuracy by using the full flow map for a few time steps: typically, using S0.66,1∘S0.33,0.66∘S0,0.33S_{0.66, 1} \circ S_{0.33, 0.66} \circ S_{0, 0.33}S0.66,1​∘S0.33,0.66​∘S0,0.33​. This needs two more feedforward passes but the resulting x1x_1x1​ is probably closer to the real solution of (1) started at x0x_0x0​.

References

Although present earlier in the litterature, the first paper to systematically distillate diffusion models was the Consistency models paper. The Flow Map Matching interprets it and generalizes it in the stochastic interpolant framework, which is the one I follow here. Very recently, the Inductive Moment Matching technique was designed to be even more efficient.
