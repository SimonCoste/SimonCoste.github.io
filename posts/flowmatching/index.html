Flow models III: Flow Matching

March 2025

Flow matching

Consider the simple ODE started from a random point and driven by the velocity field vtv_tvt​.

X˙t=vt(Xt),X0∼p0\dot{X}_t = v_t(X_t), \qquad X_0 \sim p_0X˙t​=vt​(Xt​),X0​∼p0​

We note ptp_tpt​ the probability density of XtX_tXt​ : this probability path gives a connection between the initial distribution p0p_0p0​ and the final distribution p1p_1p1​ of the random variable X1X_1X1​. It satisfies the continuity equation, also called transport equation:

∂tpt(x)=−∇⋅(vt(x)pt(x)). \partial_t p_t(x) = - \nabla \cdot (v_t(x) p_t(x)).∂t​pt​(x)=−∇⋅(vt​(x)pt​(x)).

It is is also associated with a flow: since the whole path t↦Xtt\mapsto X_tt↦Xt​ is deterministic as long as we know X0X_0X0​, we can write XtX_tXt​ as a deterministic function of X0X_0X0​.

In general, there is no simple expression for the distribution p1p_1p1​ at the end of the path. The goal of flow matching is to find a velocity field vtv_tvt​ that generates a given probability path ptp_tpt​ which starts at p0p_0p0​ (typically, a simple distribution like a Gaussian) and ends at p1p_1p1​ (typically, a complex distribution).

Conditional flows

Let (X0,X1)(X_0, X_1)(X0​,X1​) be a couple of random variables sampled from p0p_0p0​ and p1p_1p1​. We can easily define flows and velocities that transport X0X_0X0​ to X1X_1X1​. A very simple way of doing so is to simply write

Xt=(1−t)X0+tX1.X_t = (1-t)X_0 + tX_1.Xt​=(1−t)X0​+tX1​.

This can be written in terms of a family of flows noted ψty(x)=(1−t)x+ty\psi_t^y(x) = (1-t)x + tyψty​(x)=(1−t)x+ty. Clearly, the flow ψtX1\psi_t^{X_1}ψtX1​​ transports any sample X0∼p0X_0 \sim p_0X0​∼p0​ to X1X_1X1​. This flow will be called « conditional flow » since it will always end in X1X_1X1​, that is: ψ1(x∣y)=y\psi_1(x\mid y) = yψ1​(x∣y)=y. We can also compute the conditional velocity field that generates this flow:

vt(x∣X1)=ψ˙t((ψt−1(x∣X1)∣X1).v_t(x \mid X_1) = \dot \psi_t \left((\psi_t^{-1}(x \mid X_1) \mid X_1 \right).vt​(x∣X1​)=ψ˙​t​((ψt−1​(x∣X1​)∣X1​).

It is easily checked that the ODE X˙t=vt(Xt∣X1)\dot X_t = v_t(X_t \mid X_1)X˙t​=vt​(Xt​∣X1​) with initial condition X0=xX_0 = xX0​=x has solution Xt=ψt(x∣X1)X_t = \psi_t(x \mid X_1)Xt​=ψt​(x∣X1​). We will note pt(⋅∣X1)p_t(\cdot \mid X_1)pt​(⋅∣X1​) the density of this solution (which depends on X1X_1X1​), and call it the conditional path.

Suppose now that we average these conditional flows over the distribution of the endpoint X1∼p1X_1\sim p_1X1​∼p1​. This is going to be called the annealed flow, and it is defined by ψt(x)=E[ψt(x∣X1)]\psi_t(x) = \mathbb{E}[\psi_t(x \mid X_1)]ψt​(x)=E[ψt​(x∣X1​)]. Since ψ1(x∣X1)=X1\psi_1(x \mid X_1) = X_1ψ1​(x∣X1​)=X1​, this annealed flow has the right property: it bridges any starting point to a sample X1∼p1X_1 \sim p_1X1​∼p1​. But what is the velocity field that generates this annealed flow? In general, it not the annealing of vt(⋅∣X1)v_t(\cdot \mid X_1)vt​(⋅∣X1​), but rather its conditional average. We will note

pt(x)=∫pt(x∣x1)p1(x1)dx1=E[pt(x∣X1)]p_t(x) = \int p_t(x \mid x_1)p_1(x_1)dx_1 = \mathbb{E}[p_t(x \mid X_1)]pt​(x)=∫pt​(x∣x1​)p1​(x1​)dx1​=E[pt​(x∣X1​)]

the annealed probability path, ie the density of the annealed path.

The velocity field vt(x)=∫Rdvt(x∣x1)pt(x∣x1)p1(x1)pt(x)dx1v_t(x)= \int_{\mathbb{R}^d} v_t(x \mid x_1)\frac{p_t(x \mid x_1)p_1(x_1)}{p_t(x)}dx_1vt​(x)=∫Rd​vt​(x∣x1​)pt​(x)pt​(x∣x1​)p1​(x1​)​dx1​ generates the annealed flow ptp_tpt​.

Since pt(x∣x1)p1(x1)p_t(x\mid x_1)p_1(x_1)pt​(x∣x1​)p1​(x1​) is actually the joint density of XtX_tXt​ and X1X_1X1​, the formula (6) is exactly

vt(x)=E[vt(x∣X1)∣Xt=x].v_t(x) = \mathbb{E}[v_t(x \mid X_1) \mid X_t = x].vt​(x)=E[vt​(x∣X1​)∣Xt​=x].

Proof. Differentiating ptp_tpt​ in ttt yields

∂tpt(x)=∫p1(x1)∂tpt(x∣x1)dx1.\partial_t p_t(x) = \int p_1(x_1) \partial_t p_t(x\mid x_1) dx_1.∂t​pt​(x)=∫p1​(x1​)∂t​pt​(x∣x1​)dx1​.

Since the conditional probability path pt(⋅∣x1)p_t(\cdot \mid x_1)pt​(⋅∣x1​) satisfies the continuity equation with the flow vt(⋅∣X1)v_t(\cdot \mid X_1)vt​(⋅∣X1​), the RHS is equal to

−∫p1(x1)∇⋅[vt(x∣x1)pt(x∣x1)]dx1. - \int p_1(x_1)\nabla \cdot [v_t(x\mid x_1) p_t(x \mid x_1)] dx_1 .−∫p1​(x1​)∇⋅[vt​(x∣x1​)pt​(x∣x1​)]dx1​.

Swapping the divergence and the integral, this is also

−∇⋅∫p1(x1)vt(x∣x1)pt(x∣x1)dx1. - \nabla \cdot \int p_1(x_1) v_t(x \mid x_1) p_t(x \mid x_1) dx_1. −∇⋅∫p1​(x1​)vt​(x∣x1​)pt​(x∣x1​)dx1​.

We artificially add pt(x)p_t(x)pt​(x) and we get

−∇⋅[∫vt(x∣x1)pt(x∣x1)p1(x1)pt(x)dx1×pt(x)] -\nabla \cdot \left[\int v_t(x \mid x_1)\frac{p_t(x \mid x_1)p_1(x_1)}{p_t(x)}dx_1 \times p_t(x)\right]−∇⋅[∫vt​(x∣x1​)pt​(x)pt​(x∣x1​)p1​(x1​)​dx1​×pt​(x)]

which is exactly −∇⋅(vtpt)(x)-\nabla \cdot (v_t p_t)(x)−∇⋅(vt​pt​)(x), thus proving that the annealed flow ptp_tpt​ satisfies the continuity equation with the velocity field vtv_tvt​.

Learning the flow

Let uθu^\thetauθ be any parametric function (typically, a neural network). We would like to learn the velocity field vtv_tvt​ by minimizing a simple loss function, like

L⋆(θ)=∫01E[∣utθ(Xt)−vt(Xt)∣2]dt=E[∣uτθ(Xτ)−vτ(Xτ)∣2]L_\star(\theta) = \int_0^1 \mathbb{E}[|u_t^\theta(X_t) - v_t(X_t)|^2]dt = \mathbb{E}[|u_\tau^\theta(X_\tau) - v_\tau(X_\tau)|^2]L⋆​(θ)=∫01​E[∣utθ​(Xt​)−vt​(Xt​)∣2]dt=E[∣uτθ​(Xτ​)−vτ​(Xτ​)∣2]

where τ\tauτ is a uniform distribution over [0,1][0,1][0,1]. Unfortunately, even with (6), this is intractable, since we do not know how to compute ptp_tpt​ for instance. Fortunately, this loss has the same minimizers as a tractable loss.

LLL has the same minimizers as the conditional flow matching loss, L(θ)=E[∣uτθ(Xτ)−vτ(Xτ∣X1)∣2].L(\theta) = \mathbb{E}[|u^\theta_\tau(X_\tau) - v_\tau(X_\tau \mid X_1)|^2].L(θ)=E[∣uτθ​(Xτ​)−vτ​(Xτ​∣X1​)∣2].

Proof. We can ignore the fact that τ\tauτ is random and replace it with a fixed ttt, then integrate over [0,1][0,1][0,1].

Start by developing the square in L⋆(θ)L_\star(\theta)L⋆​(θ) to get E∣vτ(Xt)∣2+E∣utθ(Xt)∣2−2E⟨vτ(Xt),utθ(Xt)⟩.\mathbb{E}|v_\tau(X_t)|^2 + \mathbb{E}|u_t^\theta(X_t)|^2 - 2\mathbb{E}\langle v_\tau(X_t), u_t^\theta(X_t)\rangle.E∣vτ​(Xt​)∣2+E∣utθ​(Xt​)∣2−2E⟨vτ​(Xt​),utθ​(Xt​)⟩. The first term is a constant with respect to θ\thetaθ. Let us examine the last term, which thanks to (6) is equal to

−2E∫Rd⟨vt(Xt∣x1),utθ(Xt)⟩pt(Xt∣x1)p1(x1)pt(Xt)dx1. -2\mathbb{E}\int_{\mathbb{R}^d} \langle v_t(X_t \mid x_1), u^\theta_t(X_t) \rangle \frac{p_t(X_t \mid x_1)p_1(x_1)}{p_t(X_t)} dx_1.−2E∫Rd​⟨vt​(Xt​∣x1​),utθ​(Xt​)⟩pt​(Xt​)pt​(Xt​∣x1​)p1​(x1​)​dx1​.

Writing the expectation as an integral over xxx with density pt(x)p_t(x)pt​(x), we can simplify this into:

−2∫∫⟨vt(x∣x1),utθ(x)⟩pt(x∣x1)p1(x1)dx1dx. -2\int \int \langle v_t(x \mid x_1), u^\theta_t(x) \rangle p_t(x \mid x_1)p_1(x_1)dx_1 dx.−2∫∫⟨vt​(x∣x1​),utθ​(x)⟩pt​(x∣x1​)p1​(x1​)dx1​dx.

That is exactly

−2E⟨vt(Xt∣X1),utθ(Xt)⟩.-2\mathbb{E}\langle v_t(X_t \mid X_1), u_t^\theta(X_t)\rangle.−2E⟨vt​(Xt​∣X1​),utθ​(Xt​)⟩.

Plug this back in (14), add then substract E[∣vt(Xt∣X1)∣2]\mathbb{E}[|v_t(X_t \mid X_1)|^2]E[∣vt​(Xt​∣X1​)∣2], and conclude that

L⋆(θ)=constant+E[∣utθ(Xt)−vt(Xt∣X1)∣2]+E[∣vt(Xt∣X1)∣2]. L_\star(\theta) = \mathrm{constant} + \mathbb{E}[|u_t^\theta(X_t) - v_t(X_t \mid X_1)|^2] + \mathbb{E}[|v_t(X_t \mid X_1)|^2].L⋆​(θ)=constant+E[∣utθ​(Xt​)−vt​(Xt​∣X1​)∣2]+E[∣vt​(Xt​∣X1​)∣2].

The last term is (again!) a constant with respect to θ\thetaθ, so finally we get L⋆(θ)=L(θ)+constantsL_\star(\theta) = L(\theta) + \mathrm{constants}L⋆​(θ)=L(θ)+constants.

Everything is now tractable. We can

  • sample batches from τ,X1,Xτ\tau, X_1, X_\tauτ,X1​,Xτ​,

  • compute vtX1(x)v_t^{X_1}(x)vtX1​​(x) because we typically have access to this quantity,

  • compute the discrepancy ∣uθ(Xt)−vtX1(Xt)∣2|u^\theta(X_t) - v_t^{X_1}(X_t)|^2∣uθ(Xt​)−vtX1​​(Xt​)∣2 for all samples in the batch,

  • backpropagate the gradient of this discrepancy to update θ\thetaθ.

It is remarkable that when the conditional velocity field comes from a conditional flow as in (4), then the conditional flow matching loss (13) features the term

vt(Xt∣X1)=ψ˙t(ψt−1(Xt∣X1)∣X1)=ψ˙t(X0∣X1),v_t(X_t \mid X_1) = \dot{\psi}_t(\psi_t^{-1}(X_t \mid X_1) \mid X_1) = \dot{\psi}_t(X_0 \mid X_1),vt​(Xt​∣X1​)=ψ˙​t​(ψt−1​(Xt​∣X1​)∣X1​)=ψ˙​t​(X0​∣X1​),

and the (unconditional) velocity is

vt(x)=E[ψ˙t(X0∣X1)∣Xt=x].v_t(x) = \mathbb{E}[\dot{\psi}_t(X_0 \mid X_1) \mid X_t = x].vt​(x)=E[ψ˙​t​(X0​∣X1​)∣Xt​=x].

the last identity coming from the fact that Xt=ψt(X0∣X1)X_t = \psi_t(X_0 \mid X_1)Xt​=ψt​(X0​∣X1​). Hence, the loss becomes

L(θ)=E[∣uτθ(Xτ)−ψ˙τ(X0∣X1)∣2].L(\theta) = \mathbb{E}[|u^\theta_\tau(X_\tau) - \dot{\psi}_\tau(X_0 \mid X_1)|^2].L(θ)=E[∣uτθ​(Xτ​)−ψ˙​τ​(X0​∣X1​)∣2].

Examples of conditional flows

« Gaussian » flows

The flow (3) is a very simple example of a conditional flow: it is a straight line going from X0X_0X0​ to X1X_1X1​, often called the rectified flow. We can still keep the linearity but modulate it using conditional flows like

σt(x1)x+μt(x1) \sigma_t(x_1)x + \mu_t(x_1)σt​(x1​)x+μt​(x1​)

where σt(⋅)\sigma_t(\cdot)σt​(⋅) is a scalar function satisfying σ0(x)=1\sigma_0(x) = 1σ0​(x)=1 and μt\mu_tμt​ is a vector function satisfying μ1(x)=x\mu_1(x) = xμ1​(x)=x and μ0(x)=0\mu_0(x) = 0μ0​(x)=0. If everything here is time-differentiable, then (4) immediately gives the conditional velocity field. We first need to inverse the flow : if x=σt(x1)x0+μt(x1)x = \sigma_t(x_1)x_0 + \mu_t(x_1)x=σt​(x1​)x0​+μt​(x1​) then x0=(x−μt(x1))/σt(x1)x_0 = (x - \mu_t(x_1)) / \sigma_t(x_1)x0​=(x−μt​(x1​))/σt​(x1​). Consequently, the conditional velocity field is

vt(x∣x1)=σ˙t(x1)σt(x1)(x−μt(x1))+μ˙t(x1).v_t(x \mid x_1) = \frac{\dot{\sigma}_t(x_1)}{\sigma_t(x_1)}(x - \mu_t(x_1)) + \dot{\mu}_t(x_1).vt​(x∣x1​)=σt​(x1​)σ˙t​(x1​)​(x−μt​(x1​))+μ˙​t​(x1​).

When X0X_0X0​ is a standard Gaussian, then this XtX_tXt​ has distribution N(μt(x1),σt(x1)2Id)N(\mu_t(x_1), \sigma_t(x_1)^2 I_d)N(μt​(x1​),σt​(x1​)2Id​), hence the name conditional Gaussian flow.

Optimal transport flows

In general, we seek a probability path ptp_tpt​ governed by a velocity flow vtv_tvt​, and connecting p0p_0p0​ and p1p_1p1​. These conditions are perfectly encoded in the continuity equation ∂tpt=−∇⋅(ptvt)\partial_t p_t = -\nabla \cdot (p_t v_t)∂t​pt​=−∇⋅(pt​vt​) and the boundary conditions p0,p1p_0, p_1p0​,p1​. The optimal transport problem is to find the velocity field vtv_tvt​ that minimizes a cost, which in this case is the kinetic energy of the flow, E∫01∣vt(Xt)∣2dt\mathbb{E}\int_0^1 |v_t(X_t)|^2 dtE∫01​∣vt​(Xt​)∣2dt where XtX_tXt​ is a sample from ptp_tpt​. Formally, the problem can be stated as finding

(pOT,vOT)=arg⁡min⁡(p,v)∈C∫01∫Rd∣vt(x)∣2pt(x)dxdt(p^\mathrm{OT}, v^\mathrm{OT}) = \arg\min_{(p, v) \in \mathscr{C}} \int_0^1 \int_{\mathbb{R}^d} |v_t(x)|^2 p_t(x)dx dt (pOT,vOT)=arg(p,v)∈Cmin​∫01​∫Rd​∣vt​(x)∣2pt​(x)dxdt

where C\mathscr{C}C is the set of all probability paths and velocity fields satisfying the continuity equation and the boundary conditions. This problem is widely studied and can be solved: it is quite intuitive that the optimal flows are straight lines, so the optimal velocity fields are constants. The flow is given by

ψtOT(x)=x+t(ϕ(x)−x) \psi_t^\mathrm{OT}(x) = x + t(\phi(x) - x)ψtOT​(x)=x+t(ϕ(x)−x)

where ϕ\phiϕ is the optimal transport map from p0p_0p0​ to p1p_1p1​. The velocity field is vtOT(x)=ϕ(x)−xv^\mathrm{OT}_t(x) = \phi(x) - xvtOT​(x)=ϕ(x)−x.

This solves the unconditional OT problem, but we need conditional flows. One nice trick goes as follows: using Jensen's inequality and de-conditioning, we can write

E∫01∣vt(Xt)∣2dt=E∫01∣E[ψ˙t(Xt∣X1)∣Xt]∣2dt⩽E∫01E[∣ψ˙t(X0∣X1)∣2∣Xt]dt=E∫01E[∣ψ˙t(X0∣X1)∣2]dt=E∫01∣ψ˙t(X0∣X1)∣2dt.\begin{aligned}\mathbb{E}\int_0^1 |v_t(X_t)|^2 dt &= \mathbb{E}\int_0^1 |\mathbb{E}[\dot\psi_t(X_t \mid X_1) \mid X_t]|^2 dt \\ &\leqslant \mathbb{E}\int_0^1 \mathbb{E}[|\dot\psi_t(X_0 \mid X_1)|^2 \mid X_t] dt \\ &= \mathbb{E}\int_0^1 \mathbb{E}[|\dot\psi_t(X_0 \mid X_1)|^2] dt \\ &= \mathbb{E}\int_0^1 |\dot\psi_t(X_0 \mid X_1)|^2 dt. \end{aligned}E∫01​∣vt​(Xt​)∣2dt​=E∫01​∣E[ψ˙​t​(Xt​∣X1​)∣Xt​]∣2dt⩽E∫01​E[∣ψ˙​t​(X0​∣X1​)∣2∣Xt​]dt=E∫01​E[∣ψ˙​t​(X0​∣X1​)∣2]dt=E∫01​∣ψ˙​t​(X0​∣X1​)∣2dt.​

This last bound is the expected kinetic energy of the conditional flow over the boundary distributions X0∼p0,X1∼p1X_0 \sim p_0, X_1 \sim p_1X0​∼p0​,X1​∼p1​. For each sample x0,x1x_0, x_1x0​,x1​, we can try to find the optimal transport map γt\gamma_tγt​ which minimizes ∫01∣γ˙t∣2dt\int_0^1 |\dot\gamma_t|^2 dt∫01​∣γ˙​t​∣2dt subject to the boundary conditions γ0=x0\gamma_0 = x_0γ0​=x0​ and γ1=x1\gamma_1 = x_1γ1​=x1​. Hereagain, the solution is a straight line γt=x0+t(x1−x0)\gamma_t = x_0 + t(x_1 - x_0)γt​=x0​+t(x1​−x0​) (this can be found formally using the Euler-Lagrange condition). The conclusion of these considerations is that the (conditional) linear flow is a minimizer of a bound on the Kinetic Energy among all the conditional flows. It might not be a minimizer of the Kinetic Energy itself, but it is a good starting point.
