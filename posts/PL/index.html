Gradient descent III: SGD for Polyak-Łojasiewicz functions

October 2023

A differentiable function f:Rd→Rf:\mathbb{R}^d \to \mathbb{R}f:Rd→R is η\etaη-Polyak-Łojasiewicz[1] (η\etaη-PL for short) if it is bounded below and if for every xxx,

Fact. η\etaη-strong convex functions are η\etaη-PL.

Proof. If xxx is a minimizer of fff then for any yyy,

Flipping sides and using f(x)=inf⁡ff(x) = \inf ff(x)=inff and ∣∇f(y)/μ−μ(y−x)∣2≥0|\nabla f(y)/\sqrt{\mu} - \sqrt{\mu}(y-x)|^2\geq 0∣∇f(y)/μ​−μ​(y−x)∣2≥0 yields (1).

What is really impressive is that most results on gradient descent or SGD transfer directly from strongly convex functions to PL functions... but PL functions need not even be convex! This is a great leap forward for optimization, and it's quite a recent finding.

Two important examples of PL functions which are not convex are

  • f(x)=dist(x,S)2f(x) = \mathrm{dist}(x,S)^2f(x)=dist(x,S)2 where SSS is any set and the distance is the euclidean one.

  • f(x)=∣F(x)−y∣2f(x) = |F(x) - y|^2f(x)=∣F(x)−y∣2 when F:Rd→RdF:\mathbb{R}^d \to \mathbb{R}^dF:Rd→Rd and the smallest singular value of the Jacobien DF(x)DF(x)DF(x) is uniformly bounded away from zero, ie min⁡xσmin⁡(DF(x))>μ\min_x \sigma_{\min}(DF(x))>\muminx​σmin​(DF(x))>μ. This roughly says that fff is locally invertible, and « uniformly so ». In this case FFF is μ\muμ-PL.

These examples show that PL functions form a really wider class than (strongly) convex functions. On the other hand, they need not have a unique minimizer, which is the case for strongly convex functions. Indeed, (1) implies that any critical point (∇f(x)=0\nabla f (x) = 0∇f(x)=0) has f(x)=inf⁡ff(x) = \inf ff(x)=inff. In other words, local minima are not unique at all, but they are all global, see (b) in the Figure below[2].

Consequently, optimization algorithms will not find minimizers of fff but they will search for the minimum value of fff instead. Just as for strongly convex functions, there is a general convergence result for vanilla Gradient Descent, but I'll only present the corresponding result for SGD. Discussion

  • Minibatch SGD. In this note and the preceding one I chose the simplest SGD version, the one where fff is a sum and we choose one of the functions to estimate the full gradient. The same proof can be adapted to more complicate settings, such as mini-batches where at each step we draw a random batch of indices {i1,…,iB}\{i_1, \dotsc, i_B\}{i1​,…,iB​} uniformly at random on [m][m][m] and estimate the gradient with 1B∑k=1B∇fik(x)\frac{1}{B}\sum_{k=1}^B \nabla f_{i_k}(x)B1​∑k=1B​∇fik​​(x). The main difference will be on the definition of the discrepancy δ\deltaδ which needs to be adapted (it's straightforward).

  • Noisy SGD. The other variant of SGD is when we have access to a noisy estimate of the gradient, say for simplicity ∇f(x)+ξ\nabla f(x) + \xi∇f(x)+ξ where ξ\xiξ is an independent small Gaussian noise. I don't know if there are results for this.

  • Local PL functions. The PL assumption is considerably more realistic than strong convexity. PL functions need not be convex and need not have a unique minimizer. However, the absence of non-global local minima can be considered highly unrealistic, especially when dealing with neural networks. One possible workaround would be to replace the global PL condition with local ones. What happens when we only assume that functions are locally PL? Among many works, this paper by Sourav Chatterjee goes in this direction. He only asks that for some point xxx and some radius r>0r>0r>0, we have f(x)⩽η(x,r)r2/4f(x) \leqslant \eta(x,r)r^2/4f(x)⩽η(x,r)r2/4 where η(x,r)\eta(x,r)η(x,r) is the PL-constant of fff in the ball B(x,r)B(x,r)B(x,r). This condition implies that the gradient flow/descent on fff does stay inside B(x,r)B(x,r)B(x,r), hence our main theorem essentially applies. However, this result is only valid for non-stochastic gradient descent…

  • Neural Networks. Applying such results to neural networks remains a moot point. The main argument is that the loss landscape for NNs in the infinite width regime (and for certain initializations), the loss landscape is almost quadratic. The paper by Liu, Zhu, Belkin is a wonderful read for anyone interested in the topic. They essentially study the small singular values of the differential of a NN – these singular values are the spectral values of the Neural Tangent Kernel, and the NTK is supposed (under certain circumstances) to be… almost constant. Hence, NNs are almost PL.
