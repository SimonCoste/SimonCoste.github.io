Kesten's theorem on affine recursions

November 2023

This note is interested in the solutions of the equation

X=lawAX+B X \stackrel{\mathrm{law}}{=} AX+B X=lawAX+B

where XXX is a random variable over Rd\mathbb{R}^dRd, AAA is a random matrix and BBB a random vector, both independent of XXX. The goal is to present Kesten's theorem, which states that on certain conditions on AAA and BBB, the solution XXX must have heavy tails; that is, P(X>x)\mathbb{P}(X>x)P(X>x) decreases polynomially fast, and not exponentially fast as for Gaussian variables. This result is somehow mind-blowing in its precision, since we have access to the exact tail exponent.

A full, rigorous proof would be packed with technicalities; I'll only show a simili-proof in dimension 111, which, although not being complete, was sufficiently clear and idea-driven to convince me that Kesten's result holds. I'm also including a simili-proof of the Renewal theorem, which might be of independent interest.

One-dimensional case

For simplicity, we restrict to the case where A,BA,BA,B are independant and have a density with respect to the Lebesgue measure. In general, it is not obvious that solutions to (1) exist, but a former result by Kesten states that it is the case if E[ln⁡∣A∣]<∞\mathbb{E}[\ln |A|]<\inftyE[ln∣A∣]<∞.

Kesten's theorem (1973).

(i) Assume that A>0A>0A>0 almost surely and that there is an s>0s>0s>0 such that E[∣A∣s]=1,E[∣A∣s(ln⁡∣A∣)+]<∞,E[∣B∣s]=1.\begin{aligned}&\mathbb{E}[|A|^s]=1, &&\mathbb{E}[|A|^s (\ln |A|)_+]<\infty, &&\mathbb{E}[|B|^s]=1.\end{aligned}​E[∣A∣s]=1,​​E[∣A∣s(ln∣A∣)+​]<∞,​​E[∣B∣s]=1.​ Then, there are two constants c±c_\pmc±​ such that when x→∞x\to \inftyx→∞, P(X>x)∼c+x−s,P(X<−x)∼c−x−s.\begin{aligned}&\mathbb{P}(X>x)\sim c_+ x^{-s},&&&\mathbb{P}(X<-x) \sim c_- x^{-s}.\end{aligned}​P(X>x)∼c+​x−s,​​​P(X<−x)∼c−​x−s.​ (ii) The same result holds if AAA can take positive and negative values, and in this case c+=c−c_+ = c_-c+​=c−​.

Sketch of proof in the 1d case

I'll only sketch the proof ideas in the subcase of (i) where in addition, BBB is nonnegative. In this case, we can safely assume that XXX is nonnegative by conditionning over the set {X>0}\{X>0\}{X>0}. We set f(x)=esxP(X>ex)f(x) = e^{sx}\mathbb{P}(X > e^x)f(x)=esxP(X>ex); our final goal is to prove that f(x)f(x)f(x) converges towards some constant when x→+∞x\to+\inftyx→+∞, the −∞-\infty−∞ case being identical.

The recursion (1) shows that f(x)=esxP(AX+B>ex)f(x) = e^{sx}\mathbb{P}(AX + B > e^x)f(x)=esxP(AX+B>ex). However, if xxx is very large, we could guesse that P(AX+B>ex)\mathbb{P}(AX + B > e^x)P(AX+B>ex) is close to P(AX>ex)\mathbb{P}(AX > e^x)P(AX>ex). This is the origin of the first trick of the proof, which is to artificially write

f(x)=esx(P(AX+B>ex)−P(AX>ex))+esxP(AX>ex).\begin{aligned} f(x) &= e^{sx}(\mathbb{P}(AX+B>e^x) - \mathbb{P}(AX>e^x)) + e^{sx}\mathbb{P}(AX>e^x). \end{aligned}f(x)​=esx(P(AX+B>ex)−P(AX>ex))+esxP(AX>ex).​

Let us note ggg the first term on the right:

g(x)=esx(P(AX+B>ex)−P(AX>ex)).g(x)=e^{sx}(\mathbb{P}(AX+B>e^x) - \mathbb{P}(AX>e^x)).g(x)=esx(P(AX+B>ex)−P(AX>ex)).

For the second term, we can express it in terms of fff by using E[As]=1\mathbb{E}[A^s]=1E[As]=1:

esxP(X>ex−ln⁡A)=E[Ases(x−ln⁡(A))1X>ex−ln⁡A].\begin{aligned} e^{sx}\mathbb{P}(X>e^{x - \ln A}) &= \mathbb{E}[A^s e^{s(x-\ln(A))}\mathbf{1}_{X>e^{x - \ln A}}]. \end{aligned}esxP(X>ex−lnA)​=E[Ases(x−ln(A))1X>ex−lnA​].​

Now, here comes the second trick: the change of measure. We assumed that E[As]=1\mathbb{E}[A^s]=1E[As]=1, hence we can define a probability distribution by Es[φ]=E[Asφ]\mathbb{E}_s[\varphi] = \mathbb{E}[A^s \varphi]Es​[φ]=E[Asφ], and the expression above becomes Es[f(x−ln⁡A)]\mathbb{E}_s[f(x- \ln A)]Es​[f(x−lnA)]. Overall, we obtain that fff is a solution of the following equation: 

∀x⩾0,f(x)=g(x)+Es[f(x−ln⁡A)].\forall x \geqslant 0, \qquad f(x) = g(x) + \mathbb{E}_s[f(x - \ln A)].∀x⩾0,f(x)=g(x)+Es​[f(x−lnA)].

If μs\mu_sμs​ is the density of log⁡A\log AlogA under the measure Ps\mathbb{P}_sPs​, this equation becomes f(x)=g(x)+f⋆μ(x)f(x) = g(x) + f\star \mu(x)f(x)=g(x)+f⋆μ(x) where ⋆\star⋆ denotes the convolution operator. Such equations are called convolution equations and they can be studied using classical probability theory. The main result of Renewal Theory, exposed later below, shows that

  • if ggg satisfies some integrability condition; 

  • and if Es[ln⁡A]>0\mathbb{E}_s[\ln A]>0Es​[lnA]>0, which is the same thing as E[Asln⁡(A)]>0\mathbb{E}[A^s \ln(A)]>0E[Asln(A)]>0;

then there is only one solution to this equation, and more crucially it satisfies

lim⁡x→∞f(x)=∫0∞g(u)duEs[ln⁡A]=∫0∞g(u)duE[Asln⁡A]=:c.\lim_{x\to \infty}f(x) = \frac{\int_0^\infty g(u)du}{\mathbb{E}_s[\ln A]} = \frac{\int_0^\infty g(u)du}{\mathbb{E}[A^s \ln A]}=:c.x→∞lim​f(x)=Es​[lnA]∫0∞​g(u)du​=E[AslnA]∫0∞​g(u)du​=:c.

If E[Asln⁡A]>0\mathbb{E}[A^s \ln A]>0E[AslnA]>0, as we assumed, then c>0c>0c>0, because ggg is a nonzero positive function so its integral is nonzero. Consequently,

lim⁡x→∞esxP(X>ex)=c. \lim_{x\to \infty}e^{sx}\mathbb{P}(X>e^x) = c.x→∞lim​esxP(X>ex)=c.

This is equivalent to P(X>x)∼cx−s\mathbb{P}(X>x) \sim cx^{-s}P(X>x)∼cx−s, as requested. There is, however, a serious catch: to apply the main Renewal theorem we need to check the two conditions listed above. The second one is nothing but an assumption. However, the first one needs ggg to be "directly Riemann integrable", which is indeed very difficult to check. I won't do this part, but it is very easy to see that if BBB is positive and diffuse, then ggg is positive and continuous.

Multi-dimensional case

In the multi-dimensional case, a similar theorem holds. One can find in the litterature a host of different settings regarding the random matrix AAA and the random vector BBB; for simplicity I'll stick to the case where AAA is positive definite almost surely, BBB has all entries nonnegative almost surely, and A,BA,BA,B have densities with respect to the Lebesgue measure. The multi-dimensional analogue of E[∣A∣s]\mathbb{E}[|A|^s]E[∣A∣s] will now be

h(a)=lim⁡n→∞E[ln⁡∥A1×...×An∥a]1n\begin{aligned}h(a) = \lim_{n\to\infty}\mathbb{E}[\ln \Vert A_1 \times ... \times A_n \Vert^a]^\frac{1}{n}\end{aligned}h(a)=n→∞lim​E[ln∥A1​×...×An​∥a]n1​​

where ∥M∥\Vert M \Vert∥M∥ is the operator norm. We also introduce the Lyapounov exponent, which is the equivalent of E[ln⁡∣A∣]\mathbb{E}[\ln |A|]E[ln∣A∣]:

Kesten's theorem in any dimension. Assume that γ<0\gamma<0γ<0 and that there is an s>0s>0s>0 such that h(s)=1,E[∥A∥s(ln⁡∥A∥)+]<∞,E[∣B∣s]<∞.\begin{aligned}&h(s)=1, && \mathbb{E}[\Vert A \Vert^s (\ln \Vert A \Vert)_+]<\infty, && \mathbb{E}[|B|^s]<\infty.\end{aligned}​h(s)=1,​​E[∥A∥s(ln∥A∥)+​]<∞,​​E[∣B∣s]<∞.​ Then, solutions of (1) exist, and XXX is heavy-tailed in any direction; in other words, for any nonzero u∈Rnu\in \mathbb{R}^nu∈Rn, there is a constant c(u)>0c(u)>0c(u)>0 such that

P(⟨u,X⟩>x)∼c(u)x−s. \mathbb{P}(\langle u, X\rangle > x) \sim c(u)x^{-s}.P(⟨u,X⟩>x)∼c(u)x−s.

The proof is much more technical without any new ideas.

Renewal theory in less than one page

Let μ\muμ be a continuous probability distribution on R+\mathbb{R}_+R+​. Our goal is to show how renewal theory can be used to study convolution equations, represent their solutions with a probabilistic model, and draw consequences on their asymptotic behaviour. We are interested in finding fff such that

f(x)=g(x)+∫f(x−u)μ(du)f(x) = g(x) + \int f(x-u) \mu(du)f(x)=g(x)+∫f(x−u)μ(du)

where ggg is some function. Noting f⋆μ(x)=∫f(x−u)μ(du)f\star \mu(x) = \int f(x-u)\mu(du)f⋆μ(x)=∫f(x−u)μ(du), we recognize a convolution between the function fff and the measure μ\muμ, and the equation is simply f−μ⋆f=gf - \mu \star f = gf−μ⋆f=g. f(x)=g(x)+E[f(x−U)]. f(x) = g(x) + \mathbb{E}[f(x-U)].f(x)=g(x)+E[f(x−U)].

But if this equation is satisfied for some fff, then plugging it into f(x−U)f(x-U)f(x−U) yields

That's the key renewal theorem.
