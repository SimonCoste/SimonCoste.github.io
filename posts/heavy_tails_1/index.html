In the preceding note on heavy tails, we defined heavy-tailed distributions as distributions for which the excess function P(X>x)\mathbb{P}(X>x)P(X>x) decays polynomially fast towards zero, and not exponentially fast (as for Gaussians for example). Formally, we asked

  • How do check if their common distribution is heavy-tailed?
